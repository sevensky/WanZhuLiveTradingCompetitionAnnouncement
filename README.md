# player-match-app

A股票市场的实盘比赛报单展示系统，方便对选手每天的持仓情况进行复盘。


## Project setup
```
pnpm install
```

### Compiles and hot-reloads for development
```
pnpm run serve
```

### Compiles and minifies for production
```
pnpm run build
```

### Customize configuration
See [Configuration Reference](https://cli.vuejs.org/config/).
